챗봇 실행 방법
================================

1. 파일 복사
   - desert 폴더 전체를 새 컴퓨터로 복사
   - 폴더 구조 유지 (templates 폴더 포함)

2. Python 설치 확인
   - 명령 프롬프트에서 "python --version" 실행
   - Python 3.7 이상이 설치되어 있어야 함

3. 필요한 패키지 설치
   - 명령 프롬프트에서 다음 명령어 실행:
     pip install flask
     pip install requests

4. 실행
   - 명령 프롬프트에서 desert 폴더로 이동:
     cd desert
   - 서버 실행:
     python app_desert.py
   - 또는:
     py app_desert.py

5. 접속
   - 웹 브라우저에서 http://127.0.0.1:5000 접속

문제 해결:
- Python이 설치되지 않은 경우: https://python.org 에서 다운로드
- pip가 없는 경우: python -m ensurepip --upgrade 실행
- 포트 5000이 사용 중인 경우: 다른 포트 사용 (app_desert.py 마지막 줄 수정)

필요한 파일:
- app_desert.py
- learned_overrides.json
- answer_feedback.json
- desert_match.json
- templates/index.html
